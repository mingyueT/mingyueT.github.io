正文

忙了两天，终于在舍友、百度的帮助下，还有自己的努力下搭好了属于自己的博客。之前在阿里云申请了一个六个月的免费虚拟主机，在上面搭了一个WordPress的博客，但由于虚拟主机“保养费”太贵，光用来撘一个博客似乎太过奢侈。于是转战GitHub，搭建博客的过程中出现了太多问题，百度了许久，最后还是在舍友wzc和zk的帮助下完成搭建。深感要做好做每件事的不易。

在安装hexo时，安装了4-5次不成功，最后在知乎上找到答案：npm源有问题，将其换成taobao的cnpm源后立即解决问题，原因npm源的下载速度太慢导致下载不成功。

更换taobao的cnpm源：在电脑cmd里输入”npm install -g cnpm –registry=https://registry.npm.taobao.prg"即可，之后有关npm的命令换成用cnpm执行，你会发现下载速度大幅提升。

在用hexo搭博客时有过三次操作失误导致文件丢失，导致本地预览不了网站，第一次出现这样问题时，纠结很久，百度也找不到答案，最后重新开始把整个文件夹删除重新‘init’。之后两次有同样的问题果断重新建。
博客主题的话，十分推崇NexT，许多人都用了这个主题，简洁美观（符合码农的审美，哈哈），最主要的是部署博客，NexT这个主题在网上的教程十分详细（其他没用过，也不清楚），百度就能搜到好多！

参考博客教程：

  • hexo的next主题个性化教程:打造炫酷网站

嘿嘿，不说了，继续去布置我的博客，开始我的博客之旅~
