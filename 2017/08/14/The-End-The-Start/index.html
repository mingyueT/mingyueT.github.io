The End The Start

发表于 2017-08-14 | 分类于 start |
字数统计 204 | 阅读时长 1

想了想，还是放弃了吧，自己还有自己的路要走，，哈哈，但终究还是有些小难受。今天晚上，爸爸妈妈也说了一些关于我要认真学习的话，怎么说，可能我还是不够成熟吧，家里的负担越来越重了，爸爸的工作越来越不好找，妈妈在家里也很愁，而我却想着要玩，要谈恋爱，呵呵，，真是太不懂事了。喜欢一个人……就这么随缘吧，做一个单纯的朋友就好，哈哈，开始不知道怎么说话了，，那就不说了，，
有人说一个结束，也就是一个开始，既然其他的一事无成。
那么，还是学习吧，它应该会改变一切的，哈哈。
晚安呐~
