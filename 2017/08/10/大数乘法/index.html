大数乘法

发表于 2017-08-10 | 分类于 算法 |
字数统计 357 | 阅读时长 2

题目描述 Description

给出两个正整数A和B，计算A*B的值。保证A和B的位数不超过500位。

输出描述 Output Description

输出A*B的值

想法

模拟手工计算算法：先将两个数反转，并将字符型转成整形数组，然后按一般我们现实生活中的计算一样，从一个大数个位数乘以另一大数的所有数，再计算十位….以此类推，计算过程中注意数组元素会溢出，因为一个数组元素只放一个个位数。
